Inputs:

| Type  | Input ID | Input name | Description                                       | Delay |
|-------|----------|------------|---------------------------------------------------|-------|
| Image | 0        | src        | input                                             | no    |

Outputs:

| Type   | Output ID | Output name | Description             | Format/Resolution          | Persistent |
|--------|-----------|-------------|-------------------------|----------------------------|------------|
| Buffer | 0         | mean        | approximation of median | float                      | no         |
