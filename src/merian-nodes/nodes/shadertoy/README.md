## Shadertoy

A limited reimplementation of Shadertoy.

**Note: The Shadertoy node requires shaderc.**

Outputs:

| Type  | Output name | Description          | Format/Resolution            | Persistent |
|-------|-------------|----------------------|------------------------------|------------|
| Image | out         | the shadertoy result | `vk::Format::eR8G8B8A8Unorm` | no         |
